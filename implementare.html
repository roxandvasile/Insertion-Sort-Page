Implementare - InsertionSort

Algoritm

Pentru a sorta o matrice de dimensiune n in ordine crescatoare avem urmatorii pasi:

  1. Iteram de la arr[1] la arr[n] in matrice.
  2. Comparam elementul curent key cu predecesorul sau.
  3. Daca elementul key este mai mic decat predecesorul sau, il comparam cu elementul situat inaintea lui si mutam elementele mai mari cu o pozitie mai sus pentru a face loc pentru elementul schimbat.

Implementare in Javascript

        

        // Functia pentru sortarea matricei

          function insertionSort(arr, n) {

              let i, key, j;

              for (i = 1; i < n; i++) {
                  key = arr[i];
                  j = i - 1;
          
                  /* Mutam elementele de arr[0..i-1], care
                     sunt mai mari decat variabila key, 
                     cu o pozitie inaintea pozitiei lor curente*/
                     
                  while (j >= 0 && arr[j] > key) {
                      arr[j + 1] = arr[j];
                      j = j - 1;
                  }
                  arr[j + 1] = key;
              }
          
              return arr;
          }
